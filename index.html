Some projects

Ready2Eat

Pre-order food application in Android Studio using Firebase.
Features: choose food type, set quantity, submit order, add, delete or update food category. You can log in as a client or an admin.

Online Ticket Shop

Java project representing an online ticket shop. The interface was made in Swing. It has 5 classes 5 classes on which you can apply CRUD operations.
Features: add a ticket, choose a ticket, see available tickets, remove ticket by id, see all clients, see the tickets of a specific client.

Maze-solving Robot

Implemented using an Arduino Uno Board.
The car is powered by the remote and has to find its way through the maze, using PID controller and the distance, measured by the ultrasonic sensors, for movement control.

Dots and boxes

Python project, implementing the classic "Dots and boxes" game.
You can choose between 3 levels of difficulty, the algorithm(Min-Max algorithm or Alpha-Beta algorithm) and the player.
Interface was made using PyGame.

Image Encryptor and Template Matching

Two-part program made in C. The first part performs the encryption and decryption of an image in bmp format.
The second part recognizes handwritten digits using template-images over an image.
